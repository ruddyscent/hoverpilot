# HoverPilot

![License](https://img.shields.io/badge/license-MIT-green)

Minimal Python client to connect to RealFlight Link (TCP 18083) and inspect raw data.

## Quickstart

```bash
pip install -e .
cp .env.example .env
python -m hoverpilot.main
```
